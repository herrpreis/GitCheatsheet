Git Cheatsheet

Git Commands

# Command Meaning
1 git init Creates a new Git repository
2 git status inspects the contents of the working directory and staging area
3 git add adds files from the working directory to the staging area
4 git diff shows the difference between the working directory and the staging area
5 git commit permanently stores file changes from the staging area in the repository
6 git log shows a list of all previous commits

Git Workflow

# Step How to...
1 Create a branch Get a copy of the code files to work on
2 Add commits Editing, adding, deleting files
3 Pull Requests Request your changes to be added
4 Discuss and review ...your code with teammates
5 Deploy ...from your branch for final testing
6 Merge ...your code into the main branch
